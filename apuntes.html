Tener instalado NodeJS para ver version ejecutar node -v v8.9.4 Para empezar debemos crear un archivo package.json inicial rápido: npm init Ahora tenemos que instalar el postcss-cli en el entorno de trabajo y no global. npm install postcss-cli --save-dev Para ejecutar el comando en el entorno de trabajo ejecutamos: npx postcss-cli --version (*) Documentación PostCSS CLI: https://github.com/postcss/postcss-cli 1. Para transformar un archivo en ser ejecuta el siguiente comando: npx postcss src/css/home.css -o dist/css/home.css 2. Transformar un ves que se realice un cambio: npx postcss src/css/home.css -o dist/css/home.css -w Esto permite que si actulizamos el css de desarrollo se actulice el de producción 3. Uso de Plugins de PostCSS: npx postcss src/css/home.css -o dist/css/home.css -w -u [ -u | --use] uso de plugins de PostCSS Cambiar la ruta en nuestro index.html al archivo transformado.
